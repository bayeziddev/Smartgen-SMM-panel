Each service entry follows this structure:

| Field | Description |
|---|---|
| **Service ID** | Unique identifier for the service |
| **Rate / 1000** | Price per 1,000 units |
| **Min Order** | Minimum quantity you can order |
| **Max Order** | Maximum quantity per order |
| **Average Time** | Estimated delivery time |
| **Description** | Full service specifications |
